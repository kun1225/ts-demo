## 一、Props 型別

- React 元件本質是函式，Props 就是函式參數型別。
- 可直接寫在參數上，也可先抽成 `type`。
- 做原生元素包裝時，常用 `React.ComponentProps` 重用既有 props。
- `React.FC` 舊專案常見，但新專案通常不必特別使用。

## 二、Children 型別

- `children` 最常用 `React.ReactNode`。
- `ReactNode` 代表可渲染內容，範圍最大。
- 事件型別則常用 `React.XxxEvent<元素型別>`。

## 三、`useState` 型別應用

- 大多數情況可由初始值自動推斷。
- 初始值是 `null`、空陣列、固定字串集合時，常需要手動補泛型。
- 這是泛型在 React 中最常見的第一個實戰。

## 四、`useContext`

- 建立 context 時要先定義清楚型別。
- 若可能為空，消費端要處理 `null` 或 `undefined`。

## 五、`useRef` 與 React 事件型別

- `useRef` 常用於存取 DOM 元素。
- 也可保存不影響畫面的可變資料。
- DOM 互動與表單事件，通常要搭配對應事件型別。

## 六、`React.CSSProperties`

- 適合描述 inline style 物件型別。
- 可避免 style 欄位名稱或值寫錯。

## 總結

- React + TypeScript 的重點在 Props、children、hooks 與事件型別。
- 先掌握 `ReactNode`、`useState<T>`、`useRef<T>`，實務就夠用了。
