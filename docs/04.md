## Symbol（符號型別）

`Symbol` 是 ES6 加入的原始型別，用來建立「唯一值」。

### 重點

- 每次呼叫 `Symbol()` 都會產生不同值，每個值都是獨一無二的
- `Symbol("desc")` 的描述只方便閱讀，不影響唯一性

- 常用來當物件的特殊鍵，避免命名衝突
- 用 `Symbol` 當 key 時，不會被 `Object.keys()` 這類常見方式列出

- `Symbol.for()` 可取得全域共用的 Symbol
- TypeScript 也有 `unique symbol`，表示更嚴格的唯一符號型別

## BigInt

`BigInt` 用來表示超過 `number` 安全範圍的大整數。

### 重點

- `number` 超過 `Number.MAX_SAFE_INTEGER` 後，可能會失去精準度
- `bigint` 可以安全表示非常大的整數
- 寫法有兩種：在數字後加 `n`，或使用 `BigInt()`
- `bigint` 可以做整數運算，但不能直接和 `number` 混用
- `bigint` 的除法只保留整數，小數會被捨去
